# 材料价格与产业链利润

日期：2026-05-23  
状态：v0.1，价格数据波动较大，当前以趋势和区间为主，后续补月度价格表。

## 利润传导逻辑

电池产业链利润传导通常遵循以下路径：

1. 终端车企和储能业主压价，首先传导到电芯和系统厂。
2. 电芯厂通过年度合同、价格联动和采购议价向材料端传导。
3. 正极、电解液等材料企业承受原材料和下游压价双重波动。
4. 资源价格上涨时，低成本资源企业利润弹性最大；价格下跌时，高成本项目最先承压。
5. 设备企业不直接受材料价格影响，但受电芯厂资本开支周期影响。

## 产业链利润分配判断

| 周期阶段 | 利润更集中环节 | 原因 |
|---|---|---|
| 需求快速上行、资源短缺 | 锂资源、部分材料 | 资源涨价和库存收益 |
| 电池产能紧张 | 电芯龙头、设备企业 | 订单充足、议价较强、扩产需求旺盛 |
| 产能过剩、价格战 | 低成本电芯龙头、低成本资源 | 龙头可通过规模和客户穿越周期 |
| 储能规模化 | LFP 电芯、系统集成、PCS、温控消防 | 需求增长快，但低价竞争削弱利润 |
| 技术切换期 | 新材料、设备、头部电芯 | 新技术带来阶段性溢价 |

## 关键判断

1. 2025 年锂电材料价格整体处于低位震荡，但部分品种在四季度出现明显反弹。
2. LFP 正极材料出现价格与成本倒挂的情况，说明路线高景气不等于材料企业高利润。
3. 六氟磷酸锂价格快速反弹说明供需低库存下弹性仍强，但持续性要看扩产和下游接受度。
4. 负极材料分层明显，普通人造石墨价格竞争激烈，高倍率、快充、硅碳相关产品仍有溢价空间。
5. 产业链利润最稳定的通常不是“最热环节”，而是低成本、客户强、现金流好的头部企业。
