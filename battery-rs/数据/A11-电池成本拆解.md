## 电芯成本结构

电芯成本通常由材料成本、制造成本、折旧、人工、能源、良率损失和期间费用构成。其中材料成本占比较高，且随电池路线不同而变化。

| 成本项 | 主要内容 | 对成本影响 |
|---|---|---|
| 正极材料 | LFP、NCM、LCO、LMO 等 | 通常是电芯成本最大项之一，受锂、镍、钴、磷等影响 |
| 负极材料 | 人造石墨、天然石墨、硅碳 | 受石墨化电价、针状焦/石油焦、硅碳添加影响 |
| 电解液 | 锂盐、溶剂、添加剂 | 受六氟磷酸锂和添加剂周期影响 |
| 隔膜 | 干法/湿法基膜、涂覆 | 受良率、设备折旧和涂覆工艺影响 |
| 集流体 | 铜箔、铝箔 | 受铜铝价格和轻量化技术影响 |
| 制造费用 | 设备折旧、人工、能源、厂房 | 受规模、良率、自动化和产能利用率影响 |
| BMS/Pack | 模组、结构件、线束、热管理、BMS | 对系统成本影响明显，结构创新可降低冗余 |

## LFP 与 NMC 成本差异

LFP 的成本优势主要来自不使用镍、钴，铁和磷资源成本相对低，热稳定性更好，对热管理和安全冗余的要求较低。NMC 的优势在于能量密度较高，适合长续航和高端车型，但镍、钴成本和热管理要求更高。

2025 年 LFP 电池包平均价格较 NMC 低 40% 以上，这一差异不仅来自材料，也来自应用场景。固定式储能高度采用 LFP，对体积和重量不如乘用车敏感，因此进一步放大了 LFP 的成本优势。

## 储能系统成本结构

储能系统成本不只是电池包，还包括 PCS、BMS/EMS、热管理、消防、集装箱、升压变、土建、并网、运维和安全系统。

| 成本项 | 内容 | 尽调重点 |
|---|---|---|
| 电池舱 | 电芯、模组/Pack、箱体、热管理、消防 | 电芯价格、安全设计、循环寿命 |
| PCS | 储能变流器 | 转换效率、可靠性、并网友好性 |
| EMS/BMS | 能量管理和电池管理 | SOC/SOH 估算、均衡、安全保护 |
| 温控消防 | 液冷/风冷、消防探测和抑制 | 安全事故和保险要求 |
| 土建并网 | 基础、升压站、接入系统 | 项目周期和非技术成本 |
| 运维 | 监控、维护、容量衰减管理 | 全生命周期收益 |

## 成本判断

1. 2025 年价格下行主要由 LFP 占比提升、产能过剩、制造竞争和中国供应链规模驱动。
2. 当前低价未必可持续，因为部分正极材料企业已处于亏损状态，未来可能通过出清或涨价修复。
3. 中国成本优势显著，但海外市场价格中包含关税、物流、本地制造、合规和客户认证成本。
4. 储能系统价格下降有利于需求释放，但低价中标可能带来质量、安全和回款风险。
5. 成本拆解需要同时看“材料价格”和“系统全生命周期成本”，尤其储能不能只看初始设备价格。
