# 电池发展史时间轴

日期：2026-05-23  
状态：v0.1，用于支撑主报告第 3 章。

## 关键节点

| 时间 | 事件 | 意义 |
|---|---|---|
| 1800 年 | 伏打电堆出现 | 证明化学反应可以持续产生电流，开启现代电池史 |
| 1836 年 | 丹尼尔电池出现 | 提高电压稳定性，推动早期电报等应用 |
| 1859 年 | 铅酸电池发明 | 最早被广泛商业化的可充电电池之一，至今仍有大量应用 |
| 1866 年 | 勒克朗谢电池 | 为干电池和锌锰电池发展奠定基础 |
| 1899 年 | 镍镉电池发明 | 早期高可靠可充电电池，但存在镉污染和记忆效应 |
| 20 世纪中期 | 碱性锌锰电池普及 | 一次电池进入消费品时代 |
| 1960-1970 年代 | 镍氢电池基础研究推进 | 后来成为混合动力汽车重要电池路线 |
| 1970 年代 | 锂电池研究兴起 | 锂的低电化学电位和轻质量带来高能量密度潜力 |
| 1980 年代 | Goodenough 等推动钴酸锂正极体系 | 奠定锂离子电池高电压正极基础 |
| 1980 年代后期 | Yoshino 推动可商业化锂离子体系 | 通过碳负极等路线提升安全性和循环性 |
| 1991 年 | 索尼商业化锂离子电池 | 消费电子进入锂离子时代 |
| 1990-2000 年代 | 镍氢用于混合动力汽车 | 普锐斯等车型证明电池可深度参与汽车动力系统 |
| 2000-2010 年代 | 锂离子电池推动手机和笔记本普及 | 消费电子成为锂电规模化第一阶段 |
| 2010 年代 | 新能源汽车带动动力电池爆发 | 电池从消费电子零部件转为汽车核心部件 |
| 2015-2020 年 | 中国动力电池企业快速崛起 | 本土新能源汽车市场、政策和供应链共同推动 |
| 2020 年后 | LFP 回潮，结构创新提升系统效率 | 刀片电池、CTP、CTC 等降低系统层面短板 |
| 2021-2022 年 | 锂价暴涨，全产业链扩产 | 资源短缺和需求高增推动行业景气高点 |
| 2023-2025 年 | 价格回落、产能过剩、储能高增 | 行业从扩张期转入出清和全球化竞争期 |
| 2025 年 | 全球 EV 电池部署约 1.2TWh，LFP 超过一半 | LFP 成为全球主流路线之一，中国供应链优势强化 |
| 未来 | 钠离子、固态、液流、硅碳负极等继续演进 | 技术路线分化，场景化应用更加明显 |

## 发展史背后的产业逻辑

1. 电池每一次大规模替代，都不是单靠理论性能，而是同时满足成本、安全、寿命和制造条件。
2. 消费电子让锂离子电池完成初步规模化，新能源汽车让锂电成为战略产业，储能正在打开第二增长曲线。
3. 技术路线迁移往往与资源价格相关：锂价、钴价、镍价上升会推动低成本路线和少贵金属路线扩张。
4. 中国崛起的关键不只是补贴，更是市场规模、制造工程化、供应链完整度和企业快速迭代能力。
