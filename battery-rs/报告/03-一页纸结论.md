# 能源电池行业尽调一页纸结论

日期：2026-05-23  
版本：v1.0 底稿

## 核心判断

能源电池行业长期仍具备确定性，但本次后续研究应全力转向“电储存前沿介质与新型电池”。重点不是电网侧锂电系统集成，也不是热、氢、机械等广义储能，而是除当前主流锂离子电池之外，电能还能通过哪些电化学介质、电极/电解液、电场界面或新型电池来储存。

| 指标 | 最新口径 |
|---|---:|
| 全球电动车销量 | 2025 年超过 2000 万辆 |
| 全球 EV 电池部署量 | 2025 年约 1.2TWh |
| 全球 LFP 占 EV 电池部署量 | 2025 年超过 55% |
| 中国动力电池装车量 | 2025 年约 769.7GWh |
| 中国 LFP 装车量 | 2025 年约 625.3GWh，占约 81.2% |
| 中国新型储能累计装机 | 2025 年底约 136GW/351GWh |

| 全球新增电池储能 | 2025 年约 108GW，同比增长 40% |
| 全球储能技术结构 | 2025 年 LFP 约占电池储能部署 90% |

## 产业结论

1. 电储存前沿介质是后续第一研究方向。
2. LFP 仍是成熟主力，但新型路线会按时长、功率、安全和成本分工。
3. 宁德时代和比亚迪主导全球动力电池市场，二线企业必须证明盈利能力。
4. 材料环节路线受益不等于利润受益，产能过剩会压低加工费。
5. 锂仍是核心资源变量，但投资逻辑已转向低成本和现金流。
6. 欧盟电池法、美国 IRA/FEOC 使合规能力成为新门槛。
7. 回收长期重要，短期受退役量、LFP 经济性和金属价格制约。
8. 钙钛矿微光发电属于光伏发电/弱光能量采集，不是储能方向。
9. 固态电池长期重要，但短期以半固态和小批量示范为主，需警惕概念化。

## 优先关注

| 方向 | 关注理由 |
|---|---|
| 钠离子电池 | 最接近产业化的非锂路线 |
| 水系锌电池 | 安全、低成本，适合固定电储存 |
| 液流电池 | 长时电储存核心路线，能量和功率可分离 |
| 金属空气电池 | 铁空气、锌空气适合超长时低成本电储存 |
| 有机液流 | 分子可设计，减少金属资源依赖 |
| 超级电容/混合电容 | 高功率、短时、长寿命 |
| 龙头电芯企业 | 规模、客户、现金流、研发和海外合规能力更强 |
| 低成本锂资源 | 可穿越锂价周期 |
| 负极和石墨供应链 | 中国优势明显，海外合规价值提升 |
| 回收闭环 | 中长期受益于退役量和欧盟电池法 |
| 设备海外订单 | 海外本地化建厂带来结构性机会 |
| 固态电池材料和设备 | 长期受益于全固态产业化，但需验证良率和成本 |
| 钙钛矿微光发电 | 适合物联网和低功耗终端，属于发电侧能源采集 |

## 需要规避

1. 高成本、无现金流支撑的边际资源项目。
2. 无客户绑定、产能利用率低的电芯或材料扩产。
3. 单纯依靠低价中标的储能系统企业。
4. 只讲固态、钠电概念但无量产订单和成本验证的项目。
5. 只讲钙钛矿效率但没有稳定性、封装、良率和客户验证的项目。
6. 应收账款、存货和资本开支快速膨胀的企业。

## 最终建议

电储存项目尽调应采用“储能介质 + 电化学机理 + 循环寿命 + 安全副反应 + 材料成本 + 放大制造”六维框架。重点不是看概念是否新，而是看能否从实验室扣式电池走向 Ah/kWh/MWh 级可制造、可循环、可回收、低成本和高安全系统。
